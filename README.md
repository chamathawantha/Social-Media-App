#SocialApp - Modern Social Media UI

SocialApp is a sleek, modern social media interface built with HTML5, CSS3, and JavaScript. It features a fully responsive layout including a home feed, reels, direct messaging, and user profile management.

#Features ✨

#Home Feed:** Browse posts and create new updates with image previews.
*Interactive Polls:Create polls with dynamic options directly from the post creator.
*Reels Experience:A dedicated page for short-form video content.
*Real-time Messaging UI: A functional chat interface with conversation lists and message status.
*User Profile:View and edit personal information, bio, and social stats.
*Dark & Light Mode:Seamlessly toggle between themes for a personalized experience.
*Notification System:Built-in notification panel and unread message badges.
*Stories & Live Bar:View active user stories and live broadcasts.

#Tech Stack 🛠️

*HTML5 - Semantic structure.
*CSS3 - Custom styling, Flexbox, and Grid layouts.
*JavaScript (ES6+) - Logic for theme switching, modal handling, and UI updates.
*Font Awesome - High-quality iconography.
*Google Fonts - Utilizes the 'Nunito' font family.

#Getting Started
*Open the app:**
    Simply open `index.html` in your preferred web browser.

#File Structure 📂 

*`index.html`: The main dashboard and news feed.
*`reels.html`: Short-form video discovery page.
*`messages.html`: Direct messaging and conversation interface.
*`profile.html`: User account and personal post gallery.
*`js/`: Contains `data.js`, `app.js`, and page-specific logic.
*`css/`: Contains the styling for the entire application.
